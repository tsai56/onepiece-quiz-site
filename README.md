# 樂高® 航海王人格測驗 Prototype

此版本為純前端網站，可直接部署到 Vercel、Netlify 或 Cloudflare Pages。

## 檔案

- `index.html`：網站結構
- `styles.css`：視覺樣式
- `script.js`：題目、結果、隨機選項、推薦盒組與下載結果圖邏輯

## 本版更新

- 每次開始測驗時，各題選項會隨機排列。
- 同一輪測驗中回到上一題，選項順序會維持不變。
- 結果頁已直接加入推薦產品區塊。
- 每個結果會顯示 1 款主推盒組與 1 款延伸推薦盒組。
- 推薦產品僅顯示在結果頁，不設產品跳出按鈕。
- 下載結果圖會同步顯示主推盒組。

- 首頁與結果圖左上角已改用正式 LEGO® logo 圖檔。

## 更換臉書活動貼文網址

打開 `script.js`，修改第一行：

```js
const FACEBOOK_POST_URL = "https://www.facebook.com/";
```

將網址替換成正式臉書活動貼文網址即可。

## 部署

將整個資料夾拖曳到 Vercel Drop 即可部署。
